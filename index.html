Knowledge Base & Retrieval View

Concept Mapping

Phase AIndexing

Knowledge Base Creation Documents are ingested, chunked, converted to embeddings, and stored in a vector database for later similarity search.

Phase BRetrieval

User Prompt Processing The user's question is embedded, compared with stored vectors, and the most relevant chunks are selected.

Phase CGeneration

Answer Generation The retrieved chunks are inserted into the prompt as context. The LLM then generates a grounded answer using that context.

Key correctionImportant

The query is embedded for search. The retrieved chunks are mainly used as context for the LLM. They are not typically re-embedded for final answer generation.
